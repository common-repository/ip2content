Dynamic Content for Websites & integration of the WiredMinds LeadLab trackingcode.

== Description ==
This Plugin will change content of websites in realtime in dependence of different campaigns, branches, company names, company sizes, revenue classes and geolocations of the website visitor. The function for the dynamic content for UTM Parameter (campaigns) is free available.


== Installation ==
1. Upload "ip2c.zip" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Go to IP2C
4. Enter condition content.
5. Enter conditions.
6. Save your entries and content on webpages will be dynamic.

== Frequently Asked Questions ==
Is IP2C free?
Yes, IP2C is free to use for Campaigns / UTM parameter. Only if the attributes for the companies are used (Industry, Revenue classe etc.), theres is a token needed.

Where can i get my token?
You can get your free token from https://ip2content.wiredminds.de/licence/ and store it under licence tab.

Does IP2C use cookies?
No, the PlugIn uses no cookies at all.

Is IP2C data protection compliant?
Yes, IP2C is completly GDPR compliant. No IP adresses are stored an no personal data is used.
